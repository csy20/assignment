Solution 2: Pop up box JavaScript and data type?

JavaScript provides several types of pop-up boxes:

  • alert(): Displays a simple message and an OK button.
  • confirm(): Displays a message, an OK button, and a Cancel button. Returns true if OK is pressed, false otherwise.
  • prompt(): Displays a message, a text input field, an OK button, and a Cancel button. Returns the input value if OK is pressed, or null otherwise.

JavaScript Data Types include:

  • Primitive Types: String, Number, Boolean, Null, Undefined, Symbol, BigInt
  • Object Types: Object, Array, Function, Date, etc.
